# RAGAS_LLM_evaluation_Pytest_framewotk
Automated testing framework for RAG-based Large Language Models (LLMs) using RAGAS metrics and PyTest assertions.

Features

Evaluate RAG-based LLM responses

Automate LLM quality metrics

Support single-turn and multi-turn conversations

Generate evaluation test datasets

Assert metric thresholds with PyTest

Tech Stack

Python

PyTest

RAGAS

OpenAI API

Metrics Evaluated

Context Precision

Context Recall

Faithfulness

Answer Relevancy

Answer Correctness
